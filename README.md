# configs

All my config files. So far it only consists of: zshrc

## How to use

1. Close repo to some folder. E.g. `Users/USER-NAME/configs`
2. Import them

#### zsh

```bash
# ~/.zshrc
source /Users/phillipkingo/configs/.zshrc
```

#### git

```bash
# ~/.gitconfig
[include]
  path = ~/configs/.gitconfig
```
